# 完成的功能

1. 用户模块 (user)
    登录 (post方式)
    注册
    忘记密码和修改密码
    用户上传头像 是一个 url地址+图片名字
    用户上传文件,下载文件
    用户查看积分,头像,名称,等基本信息
    给管理员留言功能
    花钱购买积分

2. 文件资源模块 (file)
    上传文件和下载文件
    取消上传的文件
    查看文件信息(包括文件名,被下载次数,上传人,评论信息)
    评论资源功能
    添加收藏和取消收藏
    检测文件重复(根据标题)
3. 商品交换模块 (good)
    添加收藏和取消收藏
    用户上传商品和下架商品
    生成订单
    查看商品详情
    评论商品
